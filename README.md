## Crypto Convertor
**Overview**

Crypto convertor and coin Information in one place

**Requirements**

- [Node.js](https://nodejs.org/en/)
- (Optional) [Git](https://git-scm.com/)

**Setting up project**

```
git clone https://github.com/lolzivkovic/crypto-convertor.git
cd crypto-convertor/
npm install
npm run build
npm install -g serve
```
**Start project**
```
serve -s build
```
